Who? What? SantanWare?

SantanWare is an API that provides information about 1658 Santander ATMs across the UK
and 444 Santander branches. The data that we use was last updated in 2020-01-06,
and is under the open banking licence and follows the terms of use. But it begs the question...
WHY?
Why SantanWare? It's simple really. We wanted to create a easy to use API with the goal of making
banking accessible for everyone. With a few search parameters, you can find what branches have wheel chair
accessability and audio aid, aswell as the address of the ATM, the branch it belongs to and minimum withdrawl ammounts.

API reference

The SantasWhere API provides a wide range of functionality for developers, such as:

  • Retrieve data about ATMs in a town or post code
  • List all the 24 hour ATMs
  • Show all the ATMs that have wheel chair access
  • Or, combine all of the above :)

GET ATMs from Town or PostCode

For a Town:
URL parameter: town

curl -X GET "http://trenco.cs.st-andrews.ac.uk:24480/data/atms/brands/santander-uk-plc/?town=dundee"

For a PostCode:
URL parameter: postcode

curl -X GET "http://trenco.cs.st-andrews.ac.uk:24480/data/atms/brands/santander-uk-plc/?postcode=SY231DE"

GET list of 24hr ATMs

URL parameter: aoi

curl -X GET "http://trenco.cs.st-andrews.ac.uk:24480/data/atms/brands/santander-uk-plc/?aoi=true"

GET WheelChair freindly ATMs

URL parameter: wa

curl -X GET "http://trenco.cs.st-andrews.ac.uk:24480/data/atms/brands/santander-uk-plc/?wa=true"

Combined ATM Search

curl -X GET "http://trenco.cs.st-andrews.ac.uk:24480/data/atms/brands/santander-uk-plc/?town=manchester&aoi=false&wa=true"
